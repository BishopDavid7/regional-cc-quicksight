## Project summary
This Udacity Future AWS Business Intelligence Engineer Nanodegree project uses Amazon QuickSight (Q) to analyze student enrollment and course evaluation data for a community college. Deliverables include dataset preparation, QuickSight Topics & Q prompts, interactive visuals, a Scenario, and a Data Story.

## Repo contents
- `submission_document.docx` — Word file with labeled screenshots (primary submission file).
- `dataset_fields.md`, `calculated_fields.md` — dataset schema and calculated fields.
- `Q_prompts.md`, `verified_answers.md` — Q prompts and verified answers.
- `scenario_thread.md`, `data_story.md` — scenario thread and final data story.
- `manifest_sample.json` — example S3 manifest used for the attempted S3 step.
- `screenshots/` — screenshots of every project step.

## How to reproduce (high level)
1. Use Amazon QuickSight Enterprise edition with Q enabled.
2. Open the Student Enrollment Statistics sample dataset and follow the documented steps in `submission_document.docx`.
3. Use `Q_prompts.md` for exact prompts used with QuickSight Q.
